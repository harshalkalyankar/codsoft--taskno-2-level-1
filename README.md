# Kalyankar Technologies

A simple company website built using HTML, CSS, and JavaScript.

## Overview

Kalyankar Technologies is a demo software and web development company website created to showcase front-end web development skills. The project demonstrates the use of HTML, CSS, and JavaScript to create a professional multi-section business website.

## Features

* Responsive company homepage
* Navigation menu
* Services section
* About Us section
* Contact form
* JavaScript form submission alert
* Clean and beginner-friendly design

## Technologies Used

* HTML5
* CSS3
* JavaScript
